Toolbars missing

I know this seems silly but I have somehow managed to permanently hide the main toolbars in lispworks running under windows.   Under tools/preferences the Show Toolbar checkbox shows and hides the toolbars for each MDI child window, but doesn't work for the main toolbar.  Right clicking on the blank toolbar area does not give me a way to show the toolbar.  Is there a config file I can tweak or some other way that I can get the main toolbar back?  Maybe a way to reset lispworks to its original default settings?

Thanks,

Anthony

Re: Toolbars missing

Re: Toolbars missing On Mac  : Lispworks/preferences.../
Open a “Global preferences” window where there is a checkbox : “show the tools on a floating toolbar”

You can certainly find this “global preferences” window somewhere under windows

Otherwise, there is a initialization file. For me the path is : /Users/my_home/.lispworks (it’s a hidden file)
