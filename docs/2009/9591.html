Selecting an item in a list-panel

My user interface needs to select the next item in a list panel.  
Currently I'm doing:

....
(let* ((olditem (capi:choice-selected-item panel))
	((position (capi:search-for-item panel olditem)))
	((item (capi:get-collection-item table (1+ position)))))
;
(setf (capi:choice-selected-item table) item)
....

This seems a bit roundabout, as CAPI is having to do searches for the  
old and new items, but CAPI doesn't appear to provide functions  
"index of selected item" or "select nth item". Am I overlooking  
something?
