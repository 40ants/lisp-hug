start up like this:

  edi@vs2040146:/tmp$ ./foo

  Warning: Unknown map line : [ 40015000-40017000 rw-p 00014000 09:01 52387855    ]
  Warning: Unknown map line : [ 4001d000-4001f000 r-xp 00000000 09:01 52387860    ]
