The "indents.lisp" file in LW's "config" directory contains things
like this:

    (setup-indent "labels" 1 nil nil 'flet)

The last argument doesn't appear to be documented.

(BTW, SETUP-INDENT is actually EDITOR:SETUP-INDENT.)
