I just released Hunchentoot which is a LispWorks-only web server.  It
is implemented atop TBNL and therefore has a pretty small code base
although it implements HTTP/1.1 chunking, persistent connections
(keep-alive), and SSL.

More info here:

  <http://weitz.de/hunchentoot/>

Of course, this is a 0.1.0 release so it's certainly not ready for
production use yet.  I'd be interested in comments and bug reports,
though.
