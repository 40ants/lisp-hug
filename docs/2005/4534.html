I don't know if this helps, but you may want to consider OpenLisp -  
see http://www.eligis.com/

It's an implementation of ISLISP, which is more or less a Common Lisp  
subset and which should also be relatively straightforward (although  
still a lot of work) to embed in Common Lisp. OpenLisp lists ARM as  
one of the target platforms.

There are probably also Scheme implementations out there.
