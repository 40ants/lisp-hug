check-type doesn't eval for restart?

Hi all, say I've got the following code:


(defstruct foo )

(defun bar (f)
  (check-type f foo)
  'nothing-else-to-do)

(bar nil)

correct type,
and that I can supply a new value. When I choose that restart, I get a dialog
in which I can enter a form 'to be evaluated'. If I enter (make-foo), the 
error still occurs with the value (make-foo) is not of type foo. It seems that
the form is not actually being evaluated, merely read. This would seem
to be a bug. It it, or am I misunderstanding the restart? I'm on LWM 4.4.5.
