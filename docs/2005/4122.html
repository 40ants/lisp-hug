Meta key, LWM 4.4

Is there an alternative to using Escape as the Meta key in LWM4.4 ?

Re: Meta key, LWM 4.4

There are "emulation" settings that you can change to have
the editor keys behave like Emacs, or like MacOS, and 
further, whether to have no Emacs Meta key, Command
as Emacs Meta, or Alt key as Emacs Meta. 
The options are available via the red-green-blue-A key
on the toolbar, or through the Window->Window Preferences... menu. On
the drop down, they're under
Emulation.
