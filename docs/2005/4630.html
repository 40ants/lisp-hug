Announcement: Version Control for Lispworks

Version Control for Lispworks is a set of commands for those wanting to
perform version control operations from their Lispworks editor.

The beta runs on MS Windows and uses darcs.  For information on download,
use, and porting to other backends visit:

http://kantz.com/jason/programs/vc/vc-for-lw.html
