Console image not finding license file

Folks,

I saved a console version of the LW image but now when I run it  
periodically says the following:

No license entered for LispWorks.
Please run the program again with your serial number and key.
e.g. /usr/local/bin/lisp --lwlicenseserial SERIALNUMBER -- 
lwlicensekey KEY.

I run it with the license keys and then it does not ask again... for  
a while.

How do I make it pick up the license key file?

/usr/local/bin/lisp is a link to /apps/LispWorks 4.4.5/LispWorks.app/ 
Contents/MacOS/lispworks

Re: Console image not finding license file

I believe the release notes mention that LispWorks (under Mac OS X)  
will think the key is invalid if the host changes names. I've  
experienced this when my laptop is using DHCP and the DHCP server is  
issuing names in addition to IP addresses - the behavior makes  
connecting to wireless networks problematic.

Hope that helps,

-greg
