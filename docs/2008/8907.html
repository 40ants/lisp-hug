Re: Output to Listener

I don't know the answer to your specific question, but if all you want to accomplish is to see the output without manually switching, then hit WORKS >> TOOLS >> OUTPUT BROWSER and keep an output-only window open.  (It is also the blue toolbar button that looks like the logo for The Source).
