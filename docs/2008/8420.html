Gelisp: New wrapper for Gecode in Common Lisp

Hello all,

We just released a new wrapper for the Generic Constraints Development Environment (GECODE)
for Common Lisp. It is efficient, portable and provides a high-level Api.

Documentation, sources and binaries for Mac OS X, and Mac OS X Lispworks can be found at:

https://sourceforge.net/projects/gelisp/

We are currently looking for help: programmers, beta testers, reporting bugs, etc.

Cheers,
