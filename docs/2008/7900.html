lw-add-ons index page changesfor LispWorks 5.1

Adding the following updated index pages in specials.lisp will get the 
index working with LWW 5.1.  There is a warning involving 
editor::rubber-streams which I would like to know how to fix.

(defvar *index-pages*
   '(("LW" "LWRM/html/lwref-825.htm")
     ("FLI" "FLI/html/fli-187.htm")
     #+:win32
     ("COM" "COM/html/com-151.htm")
     ("CAPI" "CAPRM/html/capiref-616.htm")
     ("ED" #+:win32 "EDUG-W/html/eduser-w-162.htm")
;          #+:linux "EDUG-U/html/eduser-u-150.htm"
;          #+:mac "EDUG-M/html/eduser-m-150.htm")
     ("DLV" "DV/html/deluser-270.htm"))
   "An alist mapping shortcuts for LW documentation sections to
the relative location of their index page.")
