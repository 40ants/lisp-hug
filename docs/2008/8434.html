HSV <-> RGB conversions

With LispWorks Personal 5.1.1, (ensure-rgb (make-hsv 0.0s0 1.0s0 1.0s0))
-> #(:RGB 1.0S0 1.0S0 1.0). That's not what I expected.  Photoshop CS3,
for example, says (HSV 0 100% 100%) is RGB 255 0 0, which is full red. 
Code written from the book "Video Demystified", as well as several
on-line converters, also show red.  One such online converter is at
http://www.csgnetwork.com/csgcolorsel4.html

Too, I would expect (make-hsv (make-rgb 0.0 1.0 1.0)) to return #(:HSV
3.0 1.0 1.0) (since H goes from 0 to 5.99999).  Instead, it returns
#(:HSV 4.0 1.0 1.0).

Am I missing something?
