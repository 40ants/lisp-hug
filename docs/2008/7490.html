Re: Echo area font


On Jan 22, 2008, at 9:09 PM, Edi Weitz wrote:

> If I have an editor pane with an echo area in my application, is there
> a way to change the font of the echo area?


I don't know of a supported way, but you can get the echo pane of your  
editor pane with

capi::editor-pane-echo-area

Then try setting the font the same way you would do that for the  
editor pane.




John DeSoi, Ph.D.
