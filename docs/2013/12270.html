Problems with LWM 6.0 and "Mountain Lion"?

Hi!

I've been using LW to create demos at my university in the last year
(you've seen some of them) and I usually try to build versions for
Windows, Linux, and OS X.  Two of the programs that are heavily  used
by the students worked fine for everyone last year and now (although
unchanged) ceased to work for the Mac users.  It seems everyone who
has issues is using "Mountain Lion" (which I don't have so I can't
test myself).  One of the main problems is that text input panes no
longer work as usual.

I've been using LWM 6.0 to create these programs (I don't have a newer
version of LWM) and my OS X is ancient - 10.4.x.  Are there any known
problems with LWM 6.0 and "Mountain Lion"?  Any known reason that LW
programs which used to work on previous OS X versions shouldn't work
anymore?

Thanks,
Edi.

applications. Fortunately, that's easily solved as long as you're a
registered Mac developer and have the right certificates.)

Re: Problems with LWM 6.0 and "Mountain Lion"?

Hi,

Please check the security settings of your Mac: System Preferences - Security & Privacy - General - Allow applications downloaded from - Anywhere.

If this is set to "Mac App Store and identified developers" instead of "Anywhere," there may be problems, if I recall correctly.

Pascal
