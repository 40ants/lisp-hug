Switching floating point rounding mode

I was wondering if LispWorks has a function for switching floating
point rounding modes as specified by the IEEE 754 standard. I searched the
documentation on lispworks.com and could not find any mention of it.

In CMUCL the function call:

(set-floating-point-modes :rounding-mode :positive-infinity)

will set the floating point to round towards positive infinity. I have
an interval arithmetic library that I would like to port LispWorks.
