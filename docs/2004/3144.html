This is the sound associated with the 'Default Beep' program event in 
Windows.  To stop this event from causing any sound at all, just run 
'Control Panel -> Sounds and Audio' and select the 'Sounds' tab.  Then, 
select the 'Default Beep' program event and change the sound to '(None)' 
- it's at the top of the drop-down list.  Then click 'Apply.'

This paticular event sound drives me mad in other windows programs too. 
  I have it disabled myself.
