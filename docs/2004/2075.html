with-geometry & :titles

Greetings,

I'm trying to use (with-geometry ...) to access the geometry of a pane. 
The problem is that %x%, %width% etc. don't take into account the size 
or position of the title.  What I would like is the bounding rectangle 
of the /entire/ pane.

For example, creating a title-pane with a title but no "body".

 > (setf my-pane (make-instance 'capi:title-pane
	:title "This pane should be pretty wide"))
 > (capi:contain my-pane)
 > (capi:with-geometry my-pane %width%)
6
 > (capi:with-geometry my-pane %height%)
21

Especially troubling is this:

 > (capi:with-geometry my-pane %x%)
195

Since this seems to be not only uninformative but misinformative.

Is this a bug? Is there a way around this? Obviously layouts are able 
to correctly handle title-panes, so the bounding box must be available 
somewhere. I'm using 4.3.6 on OS X, 10.3.

Thanks,

-Chris Sims
