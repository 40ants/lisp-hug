AspectL...

Hi,

I have just released AspectL at 
http://www.common-lisp.net/project/aspectl/ which runs under LispWorks 
(tested with the Personal Edition on Mac OS X). It provides a few 
extensions to allow for aspect-oriented programming with Common Lisp, 
and I hope that some of you may find it useful.

Despite my recent nit-picking about bugs in the LispWorks MOP, it is in 
fact the only MOP implementation I have found that works reliably 
enough even for far-stretched deviations from the standard CLOS 
behavior. LispWorks is really an excellent product!


Cheers,
