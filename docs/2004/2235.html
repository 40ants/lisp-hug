Execute program under Unix PTY?

Hi!

CMUCL's EXT:RUN-PROGRAM (see 2.12 of their user manual) accepts a :PTY
keyword argument:

  "This should be one of t, nil, or a stream. If specified non-nil,
   the subprocess executes under a Unix PTY. If specified as a stream,
   the system collects all output to this pty and writes it to this
   stream. If specified as t, the process-pty slot contains a stream
   from which you can read the program's output and to which you can
   write input for the program."

Is something like this possible with LispWorks (Linux)?

(Explanation: I will need something similar to expect[1] to drive
external programs located on other hosts which are only accesible via
ssh. I have rudimentary code which does that for CMUCL using the
facilities described above but I have no idea how to do it in
LW. Ideally, this will also be portable to the Windows version of LW.)
