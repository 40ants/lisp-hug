Apparently, there were some problems with the original servers hosting 
this which were related to a building move.  You can get CL-HTTP from

ftp://ftp.se.kde.org/pub/www/servers/cl-http/

The current version is 70-190a.  If you unpack the source distribution, 
all you have to do to run it on LispWorks is load the file 
lw/start.lisp.  Take a look at lw/-read-me-.text for details.

Barry.
