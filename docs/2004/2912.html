Hi Bulent,

> is anyone using a LLGPL'ed (or BSD) ssl/https _client_ with lispworks

I've used the SSL library written by Jochen Schmidt and Wade Humeniuk
(I forgot the link, but I suppose Google will find it for you). I use
it as part of a simple-minded HTTPS server on Lispworks Pro + Windows NT.

It took me some time to get this running, mostly because I knew nothing
about SSL and I had some trouble linking it with the right dll's (for
both zlib and ssl). I also had to add a Gray streams UNREAD-CHAR method
to get things working correctly with the rest of my system.

If you're interested in my version or want to know more details, let me know.
