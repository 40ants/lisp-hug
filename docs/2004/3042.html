Changing font settings or styles-> reset cursor color

    Hi,

    When changing font settings or styles, the color of the cursor is
reset to white. In my LispWorks initialization file, I use the
following expression to set the color of the cursor:

(setf capi::*editor-cursor-color* :red)

    After doing the change, the value becomes NIL.
