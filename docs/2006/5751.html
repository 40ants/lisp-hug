Announcement: Lisp FileMaker Plug-Ins

Hi!

I suspect that most Lisp hackers don't use FileMaker (and vice versa),
but maybe some of you will find this interesting nevertheless:

FM-PLUGIN-TOOLS is a toolkit which enables you to develop FileMaker
(www.filemaker.com) plug-ins with LispWorks.  It is available from

  http://weitz.de/fm-plugin-tools/

and comes with source code for an example plug-in that, amongst other
things, reads Exif data (using Zach Beane's ZBP-EXIF library) from
pictures stored in the FileMaker database.

Also, there's already a fully-fledged plug-in available that was
written with this toolkit available from Jens Teich who sponsored the
development of FM-PLUGIN-TOOLS:

  http://jensteich.de/regex-plugin/

Have fun,
Edi.
