Announcing free KMgen

Hello,

KMgen is an ontology editor for the KM language (Knowledge Machine).

Tools used to develop KMgen: LispWorks, Foil, Java Standard 
Widget Toolkit (SWT), PostgreSQL, Emacs, Dialogue.

It is freely available from (no source code):
http://www.algo.be/ref-projects.htm#kmgen

Note:
- KMgen includes an enhanced KM emacs mode (indentation, 
coloration and completion of words in the ontology - see snapshots in the above link).
- Next release will include a better code viewer (top-right pane 
in the main window) and more.
