Re: Siglab updated for LWM 5

SigLab has been updated to run correctly on LWM 5.0.

The primary changes were related to the fact that LWM 5.0 now defaults to
single-precision floating point format, where the prior versions defaulted
to double-precision. Corrections to SigLab entailed ensuring that the
correct floating point format was stored in the C-Array objects for external
use. C-Arrays has been added to the Zip file, where it had been missing.

SigLab can be downloaded from:

    http://homepage.mac.com/dbmcclain/Mac_SigLab_Package.zip (2.6 MB)

This is a sources only package, but it contains everything you need to build
SigLab on Mac LispWorks 5.0. All foreign C functions are also included along
with a Makefile.

If you have any questions about getting it up and running, just drop me a
line at:

    dbm at refined-audiometrics dot com
