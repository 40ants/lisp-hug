No (and sorry - the doc could have been more explicit about this).

Specifically, the patch installer updates the CFBundleVersion in the
Info.plist file inside the application bundle. However it does not
touch the LispWorks image in the application bundle (or for that
matter the LispWorks with X11/Motif GUI image, which sits outside of
the application bundle, if you installed it).
