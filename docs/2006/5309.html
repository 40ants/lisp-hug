popup-menu and dialog

Is it possible to use a popup-menu on a dialog ?
By the normal way the result is a completely disabled menu as you can see in
this example : 


(let* ((menu (make-instance 'capi:menu
                :items (loop for title in '("a" "b" "c")
                            collect (make-instance 'capi:menu-item
                                            :title title))))
       (pane (make-instance 'capi:pinboard-layout
                :pane-menu menu :visible-min-height 200)))
  (capi:popup-confirmer pane "Foo"))
