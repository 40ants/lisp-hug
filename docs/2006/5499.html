CL-HTTP 70.214 s84 devo

A build of CL-HTTP 70.214 s84 devo for LispWorks is now available  
at:  http://www.cl-http.org:8001/

There are some new features and improvements available in the current  
development version which may be useful to the wider community,  
including new markup generation (XHTML 1.0 & HTML 4.0.1), SSL for the  
server, client, reverse proxy, and Web walker, new documentation  
facilities.

The release notes are available here: http://www.cl-http.org:8000/cl- 
http/release-notes.text

This version should work robustly on Lispworks.

Please report any bugs to the appropriate bug lists.

Feel free to send in your patches, new documentation, and  
contributions for inclusion in the upcoming prerelease.
