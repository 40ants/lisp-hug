Announcing Dialogue 1.0

Hello,

Dialogue is a Common Lisp tool to easy write dialogs using the 
Java Standard Widget Toolkit (SWT) and Foil (a Foreign Object 
Interface for Lisp). It provides an abstract layer over the 
controls and layout capabilities found in SWT.

This version includes many enhancements, more examples, new 
utilities and a cells editor (see screenshots).

Installation of the Java stuff and Dialogue is a snap! On 
LispWorks for Windows, at least.

Dialogue 1.0 is available from
http://www.algo.be/cl/dialogue/index.htm
