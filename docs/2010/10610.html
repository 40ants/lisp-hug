Quicklisp beta and survey

I've been working on Quicklisp, a project to make it really easy to
install and use CL libraries. Next Saturday, October 9th, I'm going to
release what I have as a public beta for anyone to try. Quicklisp works
well on LispWorks for Windows, Mac, and Linux and makes hundreds of CL
community libraries instantly accessible to LispWorks users.

I'd like to find out more about the people who might use Quicklisp, so
I've put together a 10-question Common Lisp survey. If you use CL, would
you please take a moment to fill it out?

  http://quicklisp.org/survey/
