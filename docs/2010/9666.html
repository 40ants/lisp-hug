Customize editor wrap style in LW 6

Now LW 6 has a GTK interface (nice surprise to me) I'm trying the IDE 
again but I dislike the editor's default wrap style - namely to do a 
soft return and show the excess text underneath.  I'd prefer to see that 
excess text simply hidden, but accessible by an "End of Line" command.

I have got as far as this:

   (let ((pane (editor:window-text-pane (editor:current-window))))
     (setf (capi:editor-pane-wrap-style pane) nil))

which has 2 distinct problems.

1. When the point rests on an overlong line and you do "End of Line" the 
editor window does not horizontally scroll so you cannot actually *see* 
the excess text.  The only way I have found to see it is to make the 
editor window horizontally larger.

2. You cannot include the above snippet in your .lispworks because (of 
course) there is no editor:current-window.  To work around this I have 
saved a session after evaluating the snippet in a fresh editor window 
but this is less than ideal.

Any suggestions for improvements would be most welcome.
