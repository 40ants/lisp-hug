> I guess this is a bug in save-universal-from-script?
> Workaround?

You have to make sure that the bundle exists before you call save-universal-from-script, e.g.


(save-universal-from-script (write-macos-application-bundle (namestring deliver-path)) path)

This is taken care of for you if you use the application builder.
