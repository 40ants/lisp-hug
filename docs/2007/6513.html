5.0.1: IDE tools display *very* slowly

Lispworks 5.0.1 on Windows XP. Just the other day I've noticed that the IDE tools have begun to render themselves *very* slowly. I can watch them draw themselves piece by piece; for example the Inspector tool takes over 3 seconds to complete its rendering.

I am curious to know if anyone else has experienced this behavior. Of special note is the fact that neither Lispworks 4.4 *nor* Lispworks 5.0.0 exhibit this problem. Perhaps something broke in the patch release?
