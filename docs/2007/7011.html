CAPI Property Editor

I don't think there is one, in quite the same fashion as VBasic or Allegro. And I'm about to embark on creating such a beast here too. After spending upwards of 50 hours or so with CAPI over the past week, and using pinboard-layouts and pinboard-objects, 

I think I finally have a pretty good handle on them. And it seems that is what it will take for the property editor. Using pinboards and custom pinboard-object classes allows you to do just about anything you could imagine.

BTW... coding graphics on OS X and Vista is relatively trivial compared to Win/XP. Holy Toledo!! It takes a huge amount of work to get the screen redraw flicker held down to acceptable levels. Lots of trickery and complexity involved there -- e.g., only invalidate the smallest possible regions, use XOR drawing mode as much as possible for live drawing, and definitely use some buffering (local or global). Plus, the lack of an alpha channel in the color maps is a real stinger compared to OS X.

I can only hope that XP dies a rapid death...
