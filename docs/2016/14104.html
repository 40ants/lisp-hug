Lisp shop looking for work

CSW is a Lisp shop specializing building multi-window interactive
graphical systems for data analysis and visualization.  Our
graphical-direct-manipulation UI enables users to slice and dice their
data in whatever way they deem fit.  With our graphical auditing,
replicating complex data analysis across multiple data sets becomes as
easy as doing it once and then watching the computer redo it for you.
Not a single bit of programming is required. 

The most experienced technical member at CSW has been using Common
Lisp since 1990 to do computer graphics, graphical direct
manipulation, statistical computing & graphics, and data analysis.
Currently, we use Lispworks and SBCL.

If you have a project or an instrument that requires or can benefits
from a custom graphical data exploration system, please get in touch
with CSW.
