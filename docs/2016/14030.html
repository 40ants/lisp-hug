Extra space after TEXT-INPUT-PANE in ROW-LAYOUT

I’m in Mac OS X.

I want a row-layout of text-input-panes that don’t have horizontal spacing between them, as in editing time in separate hour and minute fields (see how iCal allows you to edit event times as an example).

The following code yields fairly tight spacing, maybe not ideal, but livable:

(capi:contain (make-instance 'capi:row-layout :description (list "a" "b" "c")))

The following code does not:

(capi:contain (make-instance 'capi:row-layout :description (list (make-instance 'capi:text-input-pane :max-characters 2 :visible-max-width 20) "b" "c")))

Using :x-gap 0 doesn’t help.
