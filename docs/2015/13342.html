LW 7.0 bug (still)

So, I’m loving 7.0. But, one bug from 6 that’s still there (and I was hoping would be fixed) is the multi-line-text-input-pane. One OS X this control is pretty much unusable.

(capi:contain (make-instance 'capi:multi-line-text-input-pane :text lorem-ipsum))

Now try resizing the pane. It’s simply not possible to use this (reasonably) in an application. Is there a work-around anyone has found for it?

Just hoping that this can get fixed. ;-)

Again, awesome release!
Jeff M.
