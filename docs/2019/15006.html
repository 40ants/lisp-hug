Re: LW on Macintosh

There are rumors of a new laptop from Apple coming on Tuesday.

The single core speed for Intel mobile CPUs is only slowly rising.

The question would be, how much CPU speed can be sustained over a long period of time.
With 2.4Ghz normal speed and upto 5 Ghz Turbo ?


For an idea how the CPU speeds compare, see here:

https://browser.geekbench.com/mac-benchmarks/

Try to identify your machine and compare it to a current model.
Hoping that running for several hours does not cause thermal problems...

Generally you can expect the same speed-up for LispWorks...

Looks like the best current Macbook Pro has 1.5 the single core CPU speed.

Then see also the multicore-speeds, if that is useful for you.

There are also numbers for other computer vendors on the site.
